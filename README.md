
# Spacetravelling

Simple blog with NextJS using Prismic CMS.

## Demonstration

![Project running](/project-images/spacetravelling.gif)


## Running local

Clone project

```bash
  git clone https://github.com/Bilusca/desafio-nextjs.git
```

Install dependecies

```bash
  yarn
```

Running app

```bash
  yarn dev
```

## Stack

**Front-end:** NextJS, Sass.

**Back-end:** Prismic CMS
